# wow-agent

我们想要建立一个适合部署在企业的简易Agent教程和框架指南。wow-agent致力于在代码行数和依赖库数量之间取得均衡的最小值，用最划算的方式帮助您在本地搭建AI Agent，嵌入到您的生产工作环节中。

## 项目受众

想要在企业内部实现Agent自动办公的人。
